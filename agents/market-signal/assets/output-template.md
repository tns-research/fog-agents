## Methodology

- Source mining: `exa search` across Reddit, X / Twitter, Hacker News, IndieHackers, niche forums, Substack.
- Recency window: <N> days (default 90 for community content, 365 for blogs).
- Quote extraction: verbatim, original language, with permalink + date + upvote/karma + author signal.
- Sentiment classification: rule-based per-source (upvotes, sentiment-bearing words) + LLM check on ambiguous cases.

## Limitations

- Community content surfaces what people complain about publicly; silent-majority frustrations are under-represented.
- Sentiment is a coarse signal (positive / negative / neutral); does not capture intensity or urgency.
- Geographic skew: English-language sources dominate `exa` indexes; FR / non-English markets need explicit FR queries to avoid blind spots.
