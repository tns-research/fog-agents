## Where to look (canonical sources)

- IndieHackers: https://www.indiehackers.com/products: built-in audience filter by stage and category.
- ProductHunt: launch comments are gold for early-adopter discovery.
- Lu.ma: https://lu.ma: geography-aware event discovery.
- Read.cv & About.me: surfacingly-public profiles, often LinkedIn-adjacent and indexable.
- Substack: vertical newsletters with engaged comment sections.
- Hacker News Who's Hiring (monthly thread): real-time read on hiring patterns by vertical.

## Outreach principles (apply to every template you write)

1. **No pitch.** Frame as conversation, not sale.
2. **Specific.** Reference something they actually wrote / built / posted.
3. **Tiny ask.** "Would you trade 15 minutes for a free <thing>?" beats "Would you like to learn more about <product>?".
4. **No CTA on the first message.** The reply is the CTA.
5. **Channel-appropriate length.** Reddit DM ≤ 4 lines. LinkedIn ≤ 3 sentences. Cold email ≤ 80 words.
